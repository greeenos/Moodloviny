# Changelog

Vsechny dulezite zmeny projektu Moodloviny.

### Pridano
- Nove logo a sjednocena identita Moodloviny.
- Podpora provideru OpenAI, Gemini, Claude a OpenRouter.
- Doporucene modely primo v popupu podle zvoleneho providera.
- OpenRouter razeni modelu: nejdriv zdarma, potom placene.
- Lepsi UX popupu: rychlejsi nacitani modelu, lokalni cache, jasnejsi texty.

### Zmeneno
- Aktualizovane README pro bezne uzivatele.
- Aktualizovane modely a doporuceni k dubnu 2026.
- Vycistene historicke reference a stary branding.

### Opraveno
- Opravy textu a preklepu v popupu a dokumentaci.
- Stabilnejsi fallback seznam modelu pri vypadcich API.
